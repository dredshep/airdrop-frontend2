# How to run this:

1. [Install Nodejs](https://nodejs.org/en/download/)
2. Install Yarn
```sh
# install yarn if you don't have it
npm i -g yarn
```
3. Install dependencies
```sh
# install dependencies on first run
yarn
```

4. Run the app
```sh
# run development server
yarn dev
# build & deploy
yarn build
yarn start

# deploy on custom port
yarn start -p 12321
```
